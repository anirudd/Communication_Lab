This project performs analysis of a speech signal recorded at 8 KHz (can be changed in code). It shows the signal in time and frequency domain. The signal is passed through a low pass filter and the output is shown in the frequncy domain. 

Program File:  "lab2.m"

To run the code type the following code on the Matlab Command window:

[ record1 ] = lab2(  );

record1: contians the audio data of the recorded speech signal
